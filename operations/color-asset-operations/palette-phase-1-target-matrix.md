# 首轮 160 个 Palette 目标矩阵

> 状态：草案 v0.1（2026-05-07）
> 上游运营方案：[`./palette-market-reference-production-plan.md`](./palette-market-reference-production-plan.md)

## 1. 矩阵目的

本文用于把首轮 160 个 palette 的目标拆成可采样、可分配、可审核的子主题矩阵。

固定规则：

1. 4 个场景。
2. 每个场景 5 个子主题。
3. 每个子主题 8 个正式 palette。
4. 每个子主题建议 `6 safe + 2 expressive`。
5. 每个子主题先采样 24 套公开服饰参考，再筛到 12 个候选，最后定 8 个正式 palette。

## 7. 采样执行提醒

开始采样时，建议同步建立一份表格或后台草稿批次，至少记录：

1. 场景 ID
2. 子主题
3. 平台
4. 品牌
5. 链接
6. 采样日期
7. 主色 / 辅色 / 点缀色摘要
8. 适用品类与场景说明
9. 是否已被候选 palette 消化

如果没有这份采样台账，首轮 160 个 palette 很容易重新退回“想到哪做到哪”的状态。
